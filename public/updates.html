How We Build: Agents Talking to Agents

We're now using AI agents that collaborate with each other to build better games, faster. It started as an experiment with two agents — one brainstorming, one tearing ideas apart. The back-and-forth produced concepts neither would have reached alone.

Then we took it further. Four specialized agents now work together on every game:

Designer Agent
Proposes game concepts with mechanics, controls, and share formats. Thinks about what makes a 30-second session feel complete.
Critic Agent
Plays devil's advocate. Stress-tests ideas against our viral design principles. "Where's the emotional spike?" "Would you send this to a group chat?"
Builder Agent
Takes the refined concept and writes the entire game — HTML, CSS, JavaScript — in a single file. Knows our style guide inside out.
Playtester Agent
Reviews for bugs, balance issues, and polish gaps. Checks touch controls, share formatting, and the "open link, playing in 3 seconds" promise.

This isn't AI replacing game designers. It's AI agents having the same heated creative arguments that good design teams have — just faster and more relentlessly. The best Neon Arcade games came out of conversations where one agent said "this isn't good enough" and another rose to the challenge.

We think this multi-agent approach is the future of indie game development. Not one AI doing everything, but specialized agents collaborating — each with a different perspective, each pushing the others to do better work.

3 New Categories Launched

Three brand-new game categories are now live:

NEON CHILL — No timer, no score, just vibes. Includes Sandfall, Silk Draw, Neon Garden, and Constellation. Beautiful interactive experiences designed for relaxation.

NEON DAILY — One puzzle, every day, same for everyone. Signal Daily, Territory Daily, and Neon Grid. Streak counters and shared puzzles to keep you coming back.

NEON SOCIAL — Challenge your friends. Mimic, Veto, and Same Wave. Async multiplayer games that encode game state in the URL — no server needed.

That's 11 new games bringing our total collection to 46 playable games across 6 categories.
